Extended metadata quality assessment for recommended elements

Building on our foundational minimum mandatory work, the MSU assessed seven recommended metadata elements for compliance with our metadata criteria.

  • Green is a passing score
  • Yellow is a partial score (i.e. some but not all of the quality criteria were met)
  • Orange is a failing score (i.e. none of the quality criteria were met)

What do we mean by metadata quality?

Our recommended mandatory element assessment encompasses the following metadata quality criteria:

  • Completeness: Is the element or attribute present?
  • Accessibility: Can it be understood by a human and/or a machine?
  • Conformance: Does it meet core needs of our user community by conforming to expected standards?

Why is this important?

This information provides a more granular analysis to help the MSU strategize and track targeted remediation projects. Improving metadata quality improves discovery of and context for our online collections.

Recommended element scoring criteria

language

  • If record has a typeOfResource = "text", at least one language element must be present (Completeness)

name

  • Each name element has a name type attribute that is not blank (Accessibility)
  • Each name element has an authority type attribute that is not blank (Accessibility)
  • Each name element has a valueURI attribute that is not blank (Accessibility)
  • Each name element has a role subelement with a MARC Relator code (Accessibility)

form

  • Each form element has an authority attribute that is not blank (Accessibility)
  • Each form element has valueURI attribute that is not blank)

note

  • Each note element has a type attribute (Completeness)

tableOfContents

  • Record has no more than one tableOfContents element (Conformance)

description

  • Record has no more than one description element (Conformance)

subject

  • Each subject component has an authority attribute that is not blank (Accessibility)
  • Each subject component has valueURI attribute that is not blank)
